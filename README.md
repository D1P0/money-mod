-------------------------
Dipo's Money Mod [1.7.10]
-------------------------
Minecraft mod that adds Euro money to the game.
It contains coins from 0.01€ to 2.00€ and bills from 5.00€ to 500.00€, wallet, credit cards, ATM machine, exchanger villager..
